## 软件功能
- 支持windows可视化操作
- 可下载内核
- 同步服务器的内核列表
- 可以自定义矿池
- 同步服务器的矿池列表
- 可监控显卡状态
- 支持ssl挖矿
### 支持的内核
- NBMiner
### 支持的币种及其矿池
- ETH
  - 鱼池
  - c3pool
  - 币安
  - 自定义
- RVN
  - 鱼池
  - 自定义
