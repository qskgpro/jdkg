## 软件功能
- 支持windows可视化操作
- 可下载内核
- 同步服务器的内核列表
- 同步服务器的矿池列表
- 可监控显卡状态
- 内核反水（我自己用的别乱想）
### 支持的内核
- NBMiner
### 支持的币种及其矿池

  - 鱼池
  - c3pool
  - 币安
- RVN
